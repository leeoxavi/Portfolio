Quem Sou

Desenvolvedor Full-Stack e Analista.

Participei do Projeto Programadores Cariocas Full-Stack, uma parceria entre a Prefeitura do Rio de Janeiro e o SENAC, onde trabalhei no desenvolvimento de soluções web utilizando React. A experiência me permitiu aprimorar minhas habilidades em front-end, colaborando para criar soluções digitais acessíveis e eficazes.

Recentemente, participei do projeto CNSEG, voltado para Análise de Dados no Mercado de Seguros, onde desenvolvi habilidades em ferramentas como Pandas, NumPy e Power BI, aplicando-as para transformar dados em insights estratégicos para o setor.

Atualmente, estou envolvido no projeto Transfero, com o apoio do SENAC, onde me concentro no desenvolvimento de aplicações utilizando Django e Python.

Neste portfólio, compartilho meus projetos mais recentes, demonstrando como aplico minhas habilidades em React, Django, Python e Análise de Dados para desenvolver soluções eficientes e inovadoras.

Principais Habilidades:
